A taxi app website created using Flask. Allowing one to select their own driver, users can book a journey from a specified location to a destination using the Google Maps API. Each driver has their own unique profile, detailing their name, aggregated review score (yes this is actually calculated) and reviews (yes this is updated to show new ones). As just connoted, users can review a driver wherein they can rate them on a scale of one to five (if I remember correctly). The user can also change a plethora of details pertaining to their account: primary and secondary colours of the website (this doesn't actually change anything; however. it is still stored in a database); their username and password; other settings-related preferences - I think there was also notifaction settings as well.
